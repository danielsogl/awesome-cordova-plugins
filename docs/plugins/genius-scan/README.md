# Genius Scan

Plugin allows you to access the Genius Scan SDK core features from a Ionic application

## Installation

```bash
ionic cordova plugin add @thegrizzlylabs/cordova-plugin-genius-scan
npm install @awesome-cordova-plugins/genius-scan
```

## Supported Platforms

- Android
- iOS
